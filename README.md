## SP Toolkit

PySide6 application for interactive Raman spectrum deconvolution with `lmfit`.

## Install

```powershell
python -m pip install -e ".[dev]"
```

## Run

```powershell
sptoolkit-deconv --input-dir src/in
```

legacy module:

```powershell
python src/deconv_app.py --input-dir src/in
```
